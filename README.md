# 概要

* ビルド時のGitのコミットIDをチェックするサンプルです。

# 実行環境

* Node.js - 10.x
* Yarn -1.12.x
* Bash

# 使用ライブラリ

* TypeScript - 3.5.x
* Angular - 8.2.x

# 動作確認

## 1. サンプルのダウンロード

```
git clone git@github.com:yasu-s/ng-commit-id-check
```

## 2. パッケージインストール  

```
cd ng-commit-id-check
yarn
```

## 3. サンプルの起動  

```
yarn start
```
